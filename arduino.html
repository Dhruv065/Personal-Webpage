Arduino Project: Whack-a-Mole Game

By: Dhruv Hukkeri

Introduction

Our project was a custom-built Whack-a-Mole arcade game using an Arduino UNO. Instead of plastic moles, we used paper cups with LEDs in front of them. When the LED lit up, the player had to hit the cup. A shock sensor detected the hit, and the Arduino registered it as a hit (if within the LED’s time window) or a miss. An OLED screen displayed the score live.

Working Principle

Each cup was fitted with a balloon surface attached to a shock sensor. When hit, the sensor sent a digital signal to the Arduino UNO. If the strike matched the LED’s active time, it was recorded as a hit, otherwise as a miss.

The game included multiple difficulty levels: Noob, Intermediate, Pro, and God, where the LED timing and pause duration decreased, making the game progressively harder.

Shock Sensor Module

A shock sensor has three pins: Ground, +5V, and Signal. During a shock event, the switch closes and sends a high signal to the Arduino, enabling detection of hits.

Execution

The Arduino code randomly selected an LED to light up for 0.5 seconds. If the corresponding shock sensor detected a hit in time, a point was added. The game lasted for one minute, with score displayed on an OLED screen.

Possible extensions included randomizing LED-on time and penalizing misses with negative points.

Sample Code Snippet


int red1In = 2; 
int red1Out = 7; 
int mole = -1;
int hits = 0;

void loop() {
  mole = rand() % 4 + 1;  // Pick random mole
  digitalWrite(red1Out, HIGH);
  delay(500); // LED active
  if (digitalRead(red1In) == LOW && mole == 1) {
    hits++;
    Serial.println("Hit!");
  }
}
